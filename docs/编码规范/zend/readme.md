1.代码主题风格
====

1.1 安装主题插件
----
点击Help | Install New Software  填写Work with为[http://eclipse-color-theme.github.com/update/](http://eclipse-color-theme.github.com/update/) ，点击Add，选中Eclipse Color Theme及其子项，其他默认，然后点击Next,最后完成。
